# SwiftClock

SwiftClock provides a modern analog clock powered by SwiftUI!

## Usage

``` swift
struct ContentView: View {
    @State private var date: Date?
    
    var body: some View {
        TimelineView(.animation) { context in
            Clock(date: $date)
                .onChange(of: context.date) { oldValue, newValue in
                    date = newValue
                }
        }
        .padding()
        .background(.quinary)
    }
}
```
